# Planning Folder

This folder is the local source of truth for active work.

## Where these files come from
- Bootstrapped from this template repository (`agent-nexus`).
- Updated continuously by you and by task workflows (for example `ccpm` commands).

## Minimum file set
- `current.md`: what is actively in progress now.
- `backlog.md`: queued tasks not started yet.

## Session protocol
1. Read `current.md` first.
2. Execute or continue the top active item.
3. Sync task status (`ccpm`) and reflect changes in this folder.
4. Move completed items from `current.md` to a done section or changelog.
